About

extJWNL (Extended Java WordNet Library) is a Java API for creating, reading and updating dictionaries in WordNet format. extJWNL started from JWNL, improved it and now features support for:

  • creating, editing and writing dictionaries
  • encodings, including UTF-8
  • Java generics
  • huge dictionaries
  • instance dictionaries and static singletone dictionary
  • Maven
  • ewn: command-line tool to browse, create and edit dictionaries in WordNet format
  • and more
